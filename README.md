## Thalika Bagstore - Menjual Beragam Tas Wanita Murah dan Berkualitas

## Getting Started

First, install package.json:

```bash
npm install
```

and then run

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Stacks

- Next JS
- Tailwind CSS
- React-Redux
- Strapi
